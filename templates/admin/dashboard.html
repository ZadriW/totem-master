Integração Wake Commerce

Sincronize a biblioteca de produtos do totem com a plataforma Wake Commerce (Dental Odonto Master). Produtos existentes terão dados de catálogo atualizados; novos produtos serão adicionados com estoque zerado para configuração manual.

Use apenas em ambiente de demonstração ou quando precisar zerar o histórico. Esta ação apaga todas as vendas (e dados de clientes), apaga todo o histórico de movimentações de estoque e zera a quantidade em estoque no cadastro e em cada evento, registrando novo estoque inicial com saldo zero para reabastecer manualmente. Não é possível desfazer.
